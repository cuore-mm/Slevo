## Why

画像ビューア画面のトップバーのアイコンボタンやサムネイルを操作しても、タップや長押しが受理されたことが視覚的に分かりにくい。操作反応を明確化して誤操作不安を減らすため、即時のフィードバック表現を追加する。

## What Changes

- 画像ビューアのトップバーにある3つのアイコンボタン（戻る、共有、メニュー）に、タップ時の縮小スケールアニメーションを追加する。
- 画像ビューアのサムネイルに、タップ時の縮小スケールアニメーションを追加する。
- 画像ビューアのトップバーにある3つのアイコンボタンに、長押し時のプレーンツールチップ表示を追加する。
- フィードバック追加後も既存のタップ/長押しによる機能（画面遷移、共有、メニュー表示、サムネイル選択）は維持する。

## Capabilities

### New Capabilities

- なし

### Modified Capabilities

- `image-viewer`: ビューア画面内の操作要素に対する視覚フィードバック要件（タップ時アニメーション、長押し時ツールチップ）を追加する。

## Impact

- Affected specs: `openspec/specs/image-viewer/spec.md`
- Affected code: 画像ビューア画面の Compose UI（トップバーアイコン、サムネイルリスト）
- Dependencies: 既存の Compose Material / animation API の範囲で実装し、新規ライブラリ追加は想定しない
