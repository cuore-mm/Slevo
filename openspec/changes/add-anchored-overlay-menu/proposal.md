## Why

画像ビューアのその他メニューは標準 `DropdownMenu` の配置仕様に従っており、メニューボタンの下側に表示される。没入型のビューアUIではボタン付近に重なる表示が求められ、現在の表示位置だと視線移動が大きく操作意図と一致しにくい。

今後ほかの画面でも「アンカー上に重ねるメニュー」を再利用できるよう、ボタン座標基準で表示位置を制御できる汎用メニューコンポーネントを導入する。

## What Changes

- アンカー（ボタン）位置を基準に、メニューを上方向へ重ねて表示できる汎用オーバーレイメニューコンポーネントを追加する。
- 画面端ではみ出さないようにクランプする位置計算ロジックを実装する。
- 画像ビューアのその他メニューを新しい汎用コンポーネントへ置き換え、メニューボタン上に重なる表示へ変更する。
- 既存のメニュー項目、クリック処理、dismiss契約は維持する。

## Capabilities

### New Capabilities

- なし

### Modified Capabilities

- `image-viewer`: 画像ビューアのその他メニュー表示位置を、アンカー上への重ね表示に変更する。

## Impact

- Affected specs:
  - `openspec/specs/image-viewer/spec.md`
- Affected code:
  - 画像ビューアトップバーのメニュー表示実装
  - 共通UIコンポーネント（汎用アンカーメニュー）
- UX impact:
  - メニューボタンとメニューの視覚的連続性が向上する
  - 端末サイズ差による表示位置崩れを抑制する
