## ADDED Requirements
### Requirement: ポップアップ復帰時のアニメーション抑止
システムは画像ビューアなど別画面からスレッド画面へ戻った際、既に表示されているレスポップアップについて表示開始アニメーションを再実行してはならない（MUST NOT）。

#### Scenario: 画像ビューアから戻った場合
- **WHEN** ユーザーがレスポップアップ内の画像サムネイルから画像ビューアへ遷移し、戻る操作を行う
- **THEN** 既存のレスポップアップはアニメーションなしで表示され、状態が維持される

### Requirement: 新規ポップアップのアニメーション維持
システムは新規に追加されたレスポップアップについて、従来通り表示開始アニメーションを行う SHALL。

#### Scenario: 新しいレスポップアップを開く場合
- **WHEN** ユーザーがレス本文/ヘッダーの通常タップや返信番号タップで新しいポップアップを開く
- **THEN** 新規ポップアップは表示開始アニメーションを行う
