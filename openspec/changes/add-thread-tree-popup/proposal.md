# Change: スレッドレスのツリーポップアップ表示を追加

## Why
スレッド閲覧中に特定レスの文脈（祖先・子孫）を一括で確認できるようにし、参照の手間を減らすため。

## What Changes
- スレッド画面でレス本文/ヘッダーの通常タップ時に、該当レスの最上位祖先から子孫までのツリー全体をポップアップ表示する
- 親子関係が存在しない単独レスではツリーポップアップを表示しない
- ツリー全体ポップアップ内では、通常タップによるツリー再表示を無効化する
- 返信番号/返信元/ID/URL など既存のタップ挙動は維持する

## Impact
- Affected specs: thread-tree-popup (追加)
- Affected code: ThreadScreen のレスタップ処理、ReplyPopup / PostItem のタップ制御
