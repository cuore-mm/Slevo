## Context
ImageViewerScreen は HorizontalPager で同一レス内画像のスワイプ切替を提供しているが、一覧性やジャンプ操作が不足している。現在の表示切替はページ単位であり、サムネイル操作は存在しない。

## Goals / Non-Goals
- Goals:
  - 画像ビューア下部に同一レス内画像のサムネイル一覧を表示する
  - 表示中の画像サムネイルを中央に寄せ、視覚的に少し拡大して強調する
  - サムネイルタップで表示画像を切り替える
  - 上部バーの表示/非表示と連動してサムネイルバーも切り替える
- Non-Goals:
  - 別レスやスレッド全体の画像一覧を追加する
  - サムネイルの並び替えやフィルタ機能を追加する

## Decisions
- ImageViewerScreen 内に LazyRow を追加し、同一レス内の画像URLをサムネイルとして横並び表示する。
- 表示中のページ変更に合わせて LazyListState を `animateScrollToItem` し、中央寄せに近い位置へスクロールする。
- 選択中サムネイルはスケールを少し拡大して強調する。
- サムネイルバーは上部バーと同じ `isBarsVisible` に従って表示/非表示を切り替える。

## Alternatives considered
- ページャーの下に固定インジケーターのみを表示する: 要求されている一覧性と直接タップ遷移が満たせないため不採用。
- 別画面でサムネイル一覧を表示する: 操作が増え、スワイプ連携も複雑になるため不採用。

## Risks / Trade-offs
- サムネイル中央寄せの計算にビュー計測が必要で、実装がやや複雑になる。
  - LazyListState の layoutInfo とアイテムサイズからシンプルにオフセットを計算して最小限の実装に留める。

## Migration Plan
- ImageViewerScreen にのみ UI を追加し、既存のナビゲーション引数や画像切替ロジックは維持する。

## Open Questions
- サムネイルのサイズと間隔の具体値（UI デザインに合わせて決定する）。
