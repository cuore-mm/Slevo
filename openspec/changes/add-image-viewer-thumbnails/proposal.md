# Change: 画像ビューアのサムネイルバー追加

## Why
画像ビューアで複数画像を素早く把握・移動したいが、現在はスワイプのみで視認性と操作性が不足しているため、同一レス内画像の一覧を下部に表示して移動しやすくする必要がある。

## What Changes
- ImageViewerScreen 下部に同一レス内画像の横スクロールサムネイルを表示する
- 表示中の画像のサムネイルを中央に寄せ、少し拡大して強調表示する
- サムネイルタップで表示画像を切り替えられるようにする
- 上部バーの表示/非表示と連動してサムネイルバーも切り替える

## Impact
- Affected specs: image-viewer
- Affected code: ImageViewerScreen、サムネイル表示用の UI コンポーネント
