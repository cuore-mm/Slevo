## 1. モード判定と配色定義の整理

- [x] 1.1 画像ビューアで使用するライト/ダーク判定の参照ポイントを統一し、背景・バー・ツールチップ・前景色を返す配色定義を追加する
- [x] 1.2 ライト時は背景白/バー半透明白、ダーク時は背景黒/バー半透明黒になる色トークンを実装し、既存テーマとの整合を取る

## 2. 画像ビューアUIへの配色適用

- [x] 2.1 画像ビューア画面のルート背景にモード連動色を適用し、画面表示時に白/黒が切り替わるようにする
- [x] 2.2 トップバーとサムネイルバー背景にモード連動の半透明色を適用する
- [x] 2.3 トップバーアイコンボタンのツールチップ背景にモード連動の半透明色を適用する
- [x] 2.4 文字とアイコンの前景色を背景に応じて切り替え、ライト/ダーク双方で視認性を確保する

## 3. 検証

- [x] 3.1 ライト/ダークそれぞれで背景、バー、ツールチップ、文字/アイコン色が仕様どおりになることを確認する
